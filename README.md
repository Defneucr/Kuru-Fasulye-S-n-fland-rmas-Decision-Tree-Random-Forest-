# Kuru-Fasulye-S-n-fland-rmas-Decision-Tree-Random-Forest-
Karar Ağaçları (Decision Tree) ve Random Forest algoritmaları ile kuru fasulye (Dry Bean) veri seti sınıflandırma çalışması.
Kuru Fasulye Sınıflandırması (Decision Tree & Random Forest)

Bu proje, geometrik ve şekilsel özelliklerine göre kuru fasulye türlerini sınıflandırmak amacıyla Makine Öğrenmesi algoritmaları (Karar Ağaçları ve Rastgele Orman) kullanılarak geliştirilmiş bir Jupyter Notebook çalışmasıdır.

## Proje İçeriği ve Durumu
Bu çalışma, veri yükleme, etiket kodlama (Label Encoding), özellik seçimi, Karar Ağaçları (Decision Tree) ve Rastgele Orman (Random Forest) modellerinin kurulması adımlarını içerir.

> **Not:** Bu notebook, eğitim ve model deneme süreçlerini içeren ham çalışma dosyasıdır. Kod akışı içerisindeki veri okuma ve değerlendirme matrisi adımları, geliştirme aşamasındaki haliyle (as-is) bırakılmıştır.

## Kullanılan Teknolojiler
- Python 3
- Pandas, Matplotlib, Scikit-Learn
